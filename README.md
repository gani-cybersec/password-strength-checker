## Description
This is a beginner-friendly Python project that checks the strength of a password
based on common security rules.

## Features
- Minimum length check (8 characters)
- Uppercase letter check
- Number check
- Special character check

## How It Works
The program assigns a score based on password rules and classifies it as:
- Weak
- Medium
- Strong

## How to Run
1. Install Python
2. Run the program:
   python password_checker.py
3. Enter a test password

## Security Note
This program does not store passwords. It is for learning and testing purposes only.
